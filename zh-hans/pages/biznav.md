# 业务包导航

涂鸦现提供以下 4 个业务包，可同时使用其中的一个或者多个

## 业务包列表

| 组件                        | 内容说明                                         | 注意 |
| --------------------------- | ------------------------------------------------ | ---- |
| TuyaSmartMallBizBundle      | 提供商城功能                                     | 可选 |
| TuyaSmartActivatorBizBundle | 提供设备配网业务功能                             | 可选 |
| TuyaSmartPanelSDK           | 提供了设备控制面板的加载和控制的接口封装         | 可选 |
| TuyaCameraPanelSDK          | 提供了一系列摄像机面板相关功能                   | 可选 |
